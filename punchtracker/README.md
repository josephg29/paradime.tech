# Punch the Monkey Tracker 🐒❤️

A real-time tracking website for Punch, the world's most beloved baby Japanese macaque. Stay updated on his progress, social integration, and wellbeing at Ichikawa City Zoo.

![Punch Tracker Screenshot](https://via.placeholder.com/800x450/ff6b6b/ffffff?text=Punch+the+Monkey+Tracker)

## Features

### 📊 Real-time Status Tracking
- Current wellbeing status with visual indicators
- Detailed information about Punch's age, location, and social status
- Timeline of his journey from rejection to acceptance

### 📰 Latest News & Updates
- Breaking news about Punch's progress
- Verified information from official zoo statements
- Links to videos and photos

### 📈 Social Buzz Monitor
- Track viral engagement across platforms
- Direct links to social media discussions
- Real-time metrics for TikTok, Instagram, and Reddit

### 🏛️ Zoo Information
- Official zoo details and viewing rules
- Keeper information and statements
- Conservation context

### 🔔 Subscription System
- Email notifications for important updates
- No-spam promise
- Responsive subscription form

## Technology Stack

- **HTML5** - Semantic markup for accessibility
- **CSS3** - Modern Flexbox/Grid layout with custom properties
- **JavaScript (ES6+)** - Interactive features and animations
- **Font Awesome** - Icon library
- **Google Fonts** - Inter and Fredoka One fonts
- **Responsive Design** - Works on all devices

## Installation & Usage

1. **Clone the repository:**
   ```bash
   git clone https://github.com/scooot/punch-monkey-tracker.git
   cd punch-monkey-tracker
   ```

2. **Open in browser:**
   - Simply open `index.html` in any modern web browser
   - No build process or dependencies required

3. **Deploy to GitHub Pages:**
   - Push to GitHub repository
   - Enable GitHub Pages in repository settings
   - Site will be available at `https://scooot.github.io/punch-monkey-tracker`
